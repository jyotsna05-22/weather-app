# Weather App

This is a simple React-based web application that shows real-time weather information for any city using a weather API.

## Features

* Displays temperature, humidity, and weather conditions
* Search weather by city name
* Real-time data from a weather API
* Responsive and user-friendly interface

## Technologies Used

* React.js
* JavaScript
* HTML & CSS
* Weather API

## How to Run the Project

1. Install dependencies

   ```
   npm install
   ```
2. Start the application

   ```
   npm start
   ```
3. Open in browser

   ```
   http://localhost:3000
   ```

## Project Purpose

This project demonstrates how to build a weather forecasting web application using React and integrate external APIs to display dynamic data.
## How It Works

1. The user enters the name of a city.
2. The application sends a request to a weather API.
3. The API returns weather data such as temperature, humidity, and conditions.
4. The React app processes the response and displays the weather information on the screen.

## Example Use Case

A user can quickly check the current weather of any city to know the temperature, humidity level, and general weather conditions before planning outdoor activities or travel.
